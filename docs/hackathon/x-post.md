# Agent Adapter X Post

Built for the @OpenWallet hackathon: Agent Adapter turns any API or MCP server into a self-hosted economic agent.

Providers keep their API, wallet, keys, hosting, and pricing. The runtime discovers capabilities from OpenAPI/MCP, lets you price them locally, executes paid work through an OWS-compatible wallet stack, and supports rails like x402 and MPP.

In our demo, an agent wraps existing APIs, discovers work, fulfills tasks, and gets paid without the provider rebuilding around any one marketplace.
